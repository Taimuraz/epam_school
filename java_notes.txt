
=============================== Преобразование типов.
	- При  преобразовании большего целого типа в меньший целый тип отбрасываются старшие биты большего типа.
	- Из double -> int дробная часть отбрасывается. Без округления.
	- Очень большое double преобразуется в максимальное int число того же знака.
	- Слишком большой double при приведении к float превращается в бесконечность.
	- Любая операция над byte преобразует его в int.
	- Boxing - создание класса обертки. Инициализация с помощью примитивного типа. 
	- Unboxing - доставание из обертки значения и присваивание этого значения к примитивному типу.
 	- Классы обертки можно использовать в коллекциях. Стандартные коллекции java не поддерживают примитивные типы.
	- Преобразование из примитива в строку и обратно .parseInt() и .toString().
